# OctoAcme Templates & Checklists

This index links to all reusable templates and checklists used across the OctoAcme project management lifecycle. Each template is a Markdown file you can copy, fill in, and commit to your project repository.

---

## Available Templates

| Template | Purpose | Lifecycle Stage |
|---|---|---|
| [Project One-Pager](templates/project-one-pager-template.md) | Lightweight project charter for new initiatives | Initiation |
| [RACI Matrix](templates/raci-template.md) | Clarify responsibility assignment across roles | Planning |
| [RAID Log](templates/raid-log-template.md) | Track Risks, Assumptions, Issues, and Dependencies | Planning → Execution |
| [Decision Log](templates/decision-log-template.md) | Record key decisions with context and rationale | All stages |
| [Weekly Status Update](templates/weekly-status-update-template.md) | Structured stakeholder progress report | Execution |
| [Release Readiness Checklist](templates/release-readiness-checklist.md) | Go/no-go verification before every deployment | Release |
| [Retrospective Action Items](templates/retrospective-action-items-template.md) | Capture improvement actions from retrospectives | Retrospective |

---

## How to Use These Templates

1. Navigate to the template file linked above.
2. Copy the raw Markdown content into a new file in your project's repository (e.g., `project-docs/my-project-one-pager.md`).
3. Fill in all placeholder fields marked with `[brackets]`.
4. Commit the file and link it from your project's README or tracking issue.
